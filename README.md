# Karl Justin Rivera — Portfolio

A terminal-themed personal portfolio site built to showcase my projects as a BSIT student at Batangas State University – TNEU, Mabini Campus.

🔗 **Live site:** [add your Vercel link here once deployed]

---

## Features

- **Dark, code-editor inspired design** — sidebar navigation styled like open file tabs, section headers named like files (`projects.tsx`, `skills.json`, `availability.md`, `contact.sh`)
- **Sidebar navigation** — fixed left sidebar on desktop with active-section highlighting as you scroll; collapses into a hamburger-triggered drawer menu on mobile
- **Animated terminal intro** — a typing CLI sequence in the hero section, with reduced-motion support
- **Project showcase** — cards for each project with tech stack tags, descriptions, and live links where available
- **Built-in FAQ chatbot** — a lightweight, keyword-matching chat widget so visitors can ask about my projects, skills, availability, or how to get in touch, with no backend or API key required
- **Fully responsive** — works down to mobile screen sizes
- **Single-file build** — everything (HTML, CSS, JS) lives in one `index.html`, so it's easy to host anywhere

## Tech Stack

- HTML5 / CSS3 (custom properties, no framework)
- Vanilla JavaScript (typing animation, sidebar/menu logic, chatbot logic, scroll-spy nav)
- Fonts: Space Grotesk, IBM Plex Sans, IBM Plex Mono (Google Fonts)

## Running Locally

No build step needed — it's a static HTML file.

```bash
git clone https://github.com/Justinako/[your-repo-name].git
cd [your-repo-name]
# then just open index.html in your browser
```

Or serve it locally with any static server, e.g.:

```bash
npx serve .
```
